## 📌 Project Overview
This project is an end-to-end Machine Learning pipeline designed to predict car acceptability with professional-grade precision. By utilizing the **UCI Car Evaluation Dataset**, I developed a model that achieves a **99.13% accuracy rate** using advanced ensemble learning techniques.

This repository is officially linked with Kaggle, earning the **"Github Coder"** achievement for seamless version control and automated deployment.

## 📊 Technical Deep Dive
The model classifies cars into four categories: `unacc`, `acc`, `good`, `vgood`.

### Classification Report Snapshot:
| Category | Precision | Recall | F1-Score |
| :--- | :--- | :--- | :--- |
| **Unacceptable** | 1.00 | 1.00 | 1.00 |
| **Acceptable** | 0.99 | 0.99 | 0.99 |
| **Very Good** | 1.00 | 0.88 | 0.94 |
